# Nginx with Lua (Multi-Arch Docker Image)

## Overview
A multi-architecture Docker image for Nginx, built on `nginx:mainline-alpine` and compiled with **Lua**. 

**Pull the latest image:**
```
docker pull omaj/nginx-with-lua:latest
```

**Docker Hub Repository:**
https://hub.docker.com/r/omaj/nginx-with-lua
